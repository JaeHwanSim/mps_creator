# SW - '26 5월 월간 MPS (내 MPS)

> **2026년 5월 내 MPS — CBCT 분석 AI 고도화(segmentation·landmark) + 기준점 기반 해부학 평면(정중면·FH) 추정, 무치악 교합평면 추정 + condyle 기반 가상 교합운동 시뮬레이션 기능 개발**
>
> **입력 자료:** 4월 SW 내 MPS(Segmentation·VD Landmark 파이프라인 기준선) + 연간 SW(Smile Arch Design S/W v2.0) + 5월 사용자 지시

## Mission

4월 파이프라인 기준선 위에서 CBCT 분석 AI(segmentation·landmark)를 고도화해 기준점 기반 해부학 평면(정중면·FH)을 추정하고, 무치악 교합평면 추정과 condyle 기반 가상 교합운동 시뮬레이션 기능을 개발한다.

## 개요

4월은 두 파이프라인의 재현 가능한 코드 기준선(입출력 스펙·평가 harness·QC 자동화)을 잡았다. 5월은 그 위에서 모델을 고도화하고, 검출된 기준점으로 정중면·FH 평면 등 해부학 기준 평면을 추정한다. 이 평면과 segmentation 결과로 무치악 환자의 교합평면을 추정하고, condyle 기반으로 가상 교합운동을 시뮬레이션하는 새 기능을 개발한다.

## 목적

1. CBCT 분석 AI(**segmentation / landmark detection**)를 고도화하고, 기준점 기반 해부학 평면(**정중면·FH 평면** 등)을 추정한다.
2. 무치악 환자의 **교합평면 추정** 기능을 개발한다(치아 대신 참고선·FH 평면 기반, 러프 추정).
3. **condyle 기반 가상 교합운동 시뮬레이션** 기능을 개발한다.

## 수요자 요구 사항

- **연구·AI팀:** 고도화된 segmentation·landmark 결과와 기준점 기반 평면 산출
- **제품·SAD 2.0 통합팀:** 무치악 교합평면 추정 + 가상 교합운동 시뮬레이션 결과(샘플 E2E)
- **임상:** 교합평면·시뮬레이션의 임상 납득성(해부학 표준 부합)

## 현재 상태

### As-Is

- 4월 Segmentation·VD Landmark 재현 가능 파이프라인 기준선 확보(입출력 스펙·평가 harness·QC 자동화)
- 구개평면(ANS-PNS) primary 기준은 확보, 단 정중면·FH 평면 산출과 무치악 교합평면·가상 교합운동 시뮬레이션은 미비

### To-Be

- segmentation·landmark 고도화 + 기준점 기반 해부학 평면(정중면·FH·구개평면) 추정
- 무치악 교합평면 추정(참고선·FH 평면 기반 러프 추정)
- condyle 기반 가상 교합운동 시뮬레이션(과두경사·하악운동 파라미터)

> **참고(용어):** 정중면(Midsagittal plane, Nasion 등 정중선 통과) / FH 평면(Frankfort Horizontal, Porion-Orbitale) / 구개평면(ANS-PNS) / 교합평면(Occlusal plane) / Camper선(Ala-tragus, 무치악 교합평면 참고선) / condyle(하악과두, 과두경사=condylar inclination).

## 기대하는 결과물

1. **CBCT 분석 AI 고도화 + 해부학 평면 추정** (segmentation·landmark 고도화, 정중면·FH·구개평면 산출 모듈)
2. **무치악 교합평면 추정 기능** (참고선·잔여치조능·FH 평면 기반 러프 교합평면 후보)
3. **condyle 기반 가상 교합운동 시뮬레이션 기능** (과두경사·하악운동 파라미터, 개구/전방/측방 운동)

## Performance Objectives

### 성과 목표 1: CBCT 분석 AI 고도화 + 기준점 기반 해부학 평면 추정

> 4월 파이프라인 기준선 위에서 segmentation·landmark 검출을 고도화하고, 검출된 기준점으로 정중면·FH 평면 등 해부학 기준 평면을 추정한다.

**고정변수목표**
- Segmentation·Landmark 검출 고도화: 정확도·안정성 강화, 기준점 세트 확장(Porion/Orbitale/Nasion 등), 4월 입출력 스펙·평가 harness·QC 재사용
  - 공략전략: 4월 파이프라인·평가 harness·QC를 그대로 재사용, 경계 케이스 보정, 데이터·라벨 보강으로 고도화
- 기준점 기반 해부학 평면 추정: 정중면(Midsagittal, Nasion 등), FH 평면(Porion-Orbitale), 구개평면(ANS-PNS) 등 기준점 → 평면 변환
  - 공략전략: 기준점→평면 변환을 모듈화·단위 테스트, 평면별 정의·근거 문서화, 임상 표준(FH/MSP) 부합 검증

**변동변수목표**
- 평면 추정 안정성·일관성 평가: 좌우 대칭·재현 벤치 + 경계 케이스 리포트
  - 공략전략: 평면 일관성·좌우 대칭 오차 자동 측정, 평면 추정 안정성·일관성 합격 기준 정의

### 성과 목표 2: 무치악 교합평면 추정 + condyle 기반 가상 교합운동 시뮬레이션

> 추정한 기준점·평면과 segmentation 결과로 무치악 환자의 교합평면을 추정하고, condyle(하악과두) 기반으로 가상 교합운동을 시뮬레이션하는 기능을 개발한다.

**고정변수목표**
- 무치악 교합평면 추정: 치아가 없는 무치악 특성에 맞춰 Camper선(Ala-tragus)·잔여치조능·FH 평면 기반으로 러프한 교합평면 후보 산출
  - 공략전략: 무치악은 치아 기준이 없으므로 참고선(Camper)·FH 평면을 우선 기준으로 삼음, 추정한 평면을 기반으로 교합평면 후보 산출, 임상 납득성(샘플 근거) 검증
- condyle 기반 가상 교합운동 시뮬레이션: 과두경사(condylar inclination)·하악운동 파라미터, 개구/전방/측방 운동, 동적 교합 접촉
  - 공략전략: condyle 중심 운동 모델, 과두경사 등 파라미터 표준화, CBCT 해부학 기반 pivot 설정, 샘플 케이스로 시뮬레이션 검증

**변동변수목표**
- 교합평면·시뮬레이션 일관성 검증 리포트: 참고선·파라미터 민감도 분석 + 임상 납득성
  - 공략전략: 파라미터 민감도 벤치, 참고선·파라미터 민감도 합격 기준 정의, 임상 납득성 검증

## Risks

> 미션 전체 관점. 외부환경 1개 이상 + 내부 역량 1개 이상.

- **외부(데이터 수급·분포):** 무치악 케이스·라벨 부족 + CBCT 분포 편차(장비·FOV) → 필수 케이스 우선·범위 고정, 분포 태깅 + QC/리포트 분리
- **외부(임상 기준·참고선):** 교합평면 참고선(Camper 등)·과두 파라미터의 임상 합의 편차 → 표준 참고선·파라미터 우선 채택, 임상 검토로 보정
- **내부(재현성·민감도):** 평면 추정·시뮬레이션 파라미터 민감도로 결과 변동 → 기준점→평면 모듈 단위 테스트, 파라미터 고정·설정화, 4월 harness 재사용
- **내부(검증 기준):** 무치악 교합평면·가상 교합운동의 정답(Ground truth) 부재 → 수동 정답 소수 케이스 + 임상 납득성 검증 우선
