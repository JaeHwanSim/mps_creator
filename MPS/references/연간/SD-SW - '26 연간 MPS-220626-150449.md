# SW - '26 연간 MPS (센터 MPS — Smile Arch Design S/W v2.0)

> **SW 2026 연간 MPS**
> **핵심:** Smile Arch Design S/W v2.0 개발 — CT Landmark 기반 AI 정합 및 교합 평면 자동 보정 고도화
>
> *센터 MPS(입력 자료). 내 MPS(`MPS/created/`) 도출 시 참고.*

## Mission

**Smile Arch Design S/W v2.0 개발** — CT Landmark 기반 AI 정합 및 교합 평면 자동 보정 고도화.

## 개요

2026년은 Smile Arch Design S/W 1.1을 기반으로, **CT 랜드마크 검출 → 하악 분할 → 딥러닝 기반 자동 정합 및 교합 평면 보정 → 템플릿 생성 고도화**로 이어지는 AI 완전 자동화 기반의 **SAD 2.0**을 완성하는 해다.
5월 전시회 출시에 맞춰 **상반기에는 주요 기능을 구현**하고, **하반기에는 모델 및 기능 고도화**를 수행한다.

## 목적

1. CT 랜드마크 기반 정합·보정 자동화로 정합 정확도 향상
2. SAM 기반 VD(Vertical Dimension) 자동 보정 및 교합 평면 자동 보정 구현
3. 하악 분할(Man. Jaw segmentation) 및 딥러닝 기반 모델 생성 자동화
4. 하반기 학습 모델 고도화 및 템플릿 생성 알고리즘 개선
5. 전시회 데모용 SAD 2.0 공개 및 임상 테스트 확보

## 수요자 요구 사항

- CT 기반 자동 정합 및 교합 보정 기능
- 하악 분할을 통한 정확한 보철 설계 기준 확보
- 다양한 CT 장비(바텍, HDX 등) 지원 확대
- 임상에서 즉시 활용 가능한 자동 교합 평면 및 VD 보정 기능

## 현재 상태

### As-Is
- SAD 1.1: 자동 정합 및 하악 템플릿 완성
- AI 학습 기반 초기 segmentation 모델 확보

### To-Be
- SAD 2.0: CT 랜드마크 검출 및 VD 자동 보정 기능 내재화
- 지원 CT 확장 및 템플릿 생성 알고리즘 고도화

## 기대하는 결과물

1. **Smile Arch Design S/W v2.0** 정식 배포 버전
2. **CT Landmark Detection** 및 **SAM 기반 VD 자동 보정** 알고리즘
3. **교합 평면 자동 보정 모듈**
4. **Template 생성 고도화 알고리즘**
5. **AI 학습 데이터셋 및 고도화 모델**

---

## Performance Objectives

### 성과 목표 1: 상반기 전시회용 SAD 2.0 기능 통합 버전

> CT 기반 Landmark 검출, 하악 분할, VD 자동 보정 및 교합 평면 자동 보정 기능을 SAD 2.0에 통합하여 전시회용 데모 버전을 개발한다.

**고정변수목표 1: SAD 2.0 전시회 버전 (CT Landmark, VD 보정, 교합 평면 자동화 포함)**
- 공략전략:
  - CT Landmark 검출 AI 모델 학습 (바텍·HDX 데이터 기반)
  - 하악 segmentation 및 Man. Jaw 분할 기능 구현
  - SAM 기반 VD 자동 보정 알고리즘 및 UI 개발
  - 교합 평면 자동 보정 로직 통합
  - 전시회용 데모 버전 검증 및 안정화

**변동변수목표 1: AI 모델 학습 정확도 90% 이상 확보**
- 공략전략:
  - Landmark dataset 정제 및 augmentation 수행
  - 다양한 CT 장비별 입력 데이터 정규화

**Risk**
- **외부(일정):** 전시회 일정 내 완성도 확보 필요 → MVP 기능 기준으로 단계적 통합
- **내부(정확도):** AI 모델 정확도 부족 → 데이터셋 확장 및 반복 학습 주기 단축

### 성과 목표 2: 하반기 SAD 2.0 기능 고도화 및 지원 CT 확장

> 상반기 개발된 모델과 기능을 고도화하여 상용 수준의 SAD 2.0을 완성하고, 지원 CT 장비를 바텍, HDX 외 타 브랜드로 확장한다.

**고정변수목표 2: SAD 2.0 고도화 버전 및 Template 생성 알고리즘 개선**
- 공략전략:
  - 상반기 학습 모델 리파인 및 CT 지원 확장
  - 템플릿 생성 알고리즘 고도화 (곡률 기반 형태 적합성 향상)
  - Template 생성 속도 개선 (기존 대비 30% 단축)
  - 임상 테스트를 통한 정합 정확도 및 VD 보정 성능 검증

**변동변수목표 2: 모델 성능 검증 및 다장비 테스트 리포트**
- 공략전략:
  - 다기관 CT 데이터 검증 (국내/글로벌 법인 연계)
  - 성능 편차 분석 및 보정 로직 개선

**Risk**
- **외부(데이터):** 협력 CT 제조사 데이터 제공 지연 가능성 → 사전 NDA 및 데이터셋 병행 확보
- **내부(일정):** 기능 복잡도 증가로 일정 지연 위험 → 상반기/하반기 기능 분리 개발
- **내부(UI/UX):** UI/UX 복잡화 → Template 중심 직관적 인터페이스 유지

## 📌 요약

- **상반기:** CT Landmark 검출 + 하악 분할 + VD/교합 자동 보정 기능 개발 → 전시회 공개
- **하반기:** 딥러닝 모델 고도화 + 지원 CT 확장 + 템플릿 생성 알고리즘 개선
- **결과물:** SAD 2.0 정식 버전 및 AI 기반 자동 정합 완성
